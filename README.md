# E-Cart (Spring Boot)

A simple e-commerce application built using Spring Boot that supports
product management, image upload, and basic cart functionality.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL
- REST APIs

## Features
- Product CRUD operations
- Product image upload
- RESTful APIs
- Order creation and order management
- Environment-based configuration

## How to Run
```bash
./mvnw spring-boot:run
